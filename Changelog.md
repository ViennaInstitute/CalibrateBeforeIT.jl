## Project Overview

This changelog tracks modifications to CalibrateBeforeIT.jl, a Julia
package that provides calibration scripts for the BeforeIT.jl
agent-based macroeconomic model across the 27 EU member countries.

Recent work has included:
- Improved calibration robustness for multi-country support
- Enhanced data downloading and preprocessing functionality
- Various bug fixes and optimizations
- Quarterly time series support with fallback to seasonally adjusted data

For detailed usage instructions, see [README.md](README.md).
